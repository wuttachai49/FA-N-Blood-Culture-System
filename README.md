## 🔬 Detection Principles

### Bacterial Growth Detection
The FA-N Series uses **non-invasive colorimetry** to monitor microbial growth:

1. **Microorganism Growth** → Bacteria produce CO₂
2. **pH Change** → Acidic byproducts lower pH
3. **Chromogenic Reaction** → pH-sensitive film color changes (dark → light)
4. **Optical Detection** → Photodetector measures reflected light
5. **Algorithm Analysis** → System determines positive/negative result

### Five Detection Algorithms

- **Early Acceleration Method** — 1h 20min (short lag phase organisms)
- **Early Rate Method** — 1 hour (delayed bottles in growth phase)
- **Threshold Method** — 1h 40min (absolute reflectance detection)
- **Normal Rate Method** — 3h 40min (standard growth patterns)
- **Slow Growth Algorithm** — 6-7+ hours (fastidious and slow-growing bacteria)
