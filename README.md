# FA-N Series Automated Blood Culture System

## Overview
Comprehensive information hub for the FA-N Series Automated Blood Culture System - Fast, Accurate, New Generation blood culture detection.

## Key Features
- **Early Detection**: Results from 1 hour using advanced algorithms
- **Three Models**: FA-N60, FA-N120, FA-N240 with scalable capacity (60-480 bottles)
- **Advanced Detection**: Colorimetry-based optical detection with intelligent algorithms
- **Clinical Benefits**: Supports sepsis bundle completion and rapid bloodstream infection diagnosis
- **Automation**: Barcode-driven workflow with automatic sample volume detection (N120/N240)

## System Models
- **FA-N60**: 60 bottles, manual sample volume entry, ideal for small labs
- **FA-N120**: 120 bottles, automatic sample detection, ideal for medium hospitals
- **FA-N240**: 240 bottles, automatic sample detection, ideal for large hospitals

## Supported Culture Types
- Aerobic Cultures (MR-AC)
- Anaerobic Cultures
- Fungal Cultures
- Pediatric Cultures (MR-PC)

## Detection Algorithms
Multiple sophisticated algorithms for different organism growth patterns:
- Early Acceleration Method: 1h 20min
- Early Rate Method: 1h
- Threshold Method: 1h 40min
- Rate Method (Normal): 3h 40min
- Slow Growth Algorithm (specialized for slow-growing organisms)

## Clinical Applications
Designed for rapid detection of:
- Sepsis diagnosis
- Bloodstream infections (BSI)
- Catheter-related bloodstream infections (CRBSI)
- Support for Sepsis 3.0 Bundle implementation

## Specifications
- **Incubation Temperature**: 37°C ± 0.5°C
- **Detection Method**: Non-invasive colorimetry with photodetector
- **Sample Volume Range**: 5-10 mL (optimal)
- **Data Integration**: USB interface for LIS/Middleware connectivity
